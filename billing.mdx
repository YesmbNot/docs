## Credit System

ApiTwitter uses a **pay-per-use** credit system. You buy credits and each API call deducts a set amount.

| Key Info | |
|----------|---|
| **Exchange rate** | 1 USD = 100,000 credits |
| **Minimum top-up** | $10 (1,000,000 credits) |
| **Expiration** | Credits never expire |
| **Payment methods** | Stripe (card) / CryptoCloud (crypto) |

## Pricing Tiers

| Tier | Cost per Call | USD per 1K Calls | Endpoints |
|------|-------------|-------------------|-----------|
| **Session** | 5 credits | $0.05 | Session verify |
| **Read (data)** | 14 credits | $0.14 | Tweets, followers, timelines, lists, topics, communities, bookmarks |
| **Read (profiles)** | 16 credits | $0.16 | User profile by name, user by ID, batch users |
| **Search** | 225 credits | $2.25 | Tweet search |
| **Write** | 225 credits | $2.25 | Create/delete tweets, like, follow, retweet, bookmark, pin, DM, list management, community, topic |

## Billing Rules

<Info>
- Credits are deducted **before** the request is processed
- Failed requests (4xx/5xx from our platform) are **not charged**
- You receive a `402 Payment Required` if your balance is insufficient
</Info>

## Top Up

1. Go to [Dashboard → Billing](https://www.apitwitter.com/billing)
2. Select an amount ($10, $25, $50, $100, $250, or custom)
3. Choose **Pay with Card** (Stripe) or **Pay with Crypto** (CryptoCloud)
4. Credits are added instantly after payment confirmation

## Billing FAQ

<AccordionGroup>
  <Accordion title="Do credits expire?">
    No. Credits purchased on ApiTwitter never expire. You can use them at any time.
  </Accordion>
  <Accordion title="What payment methods are accepted?">
    ApiTwitter accepts credit/debit cards via Stripe and cryptocurrency via CryptoCloud. The minimum top-up is $10.
  </Accordion>
  <Accordion title="Am I charged for failed requests?">
    No. Failed requests that return 4xx or 5xx errors from the ApiTwitter platform are not charged. Credits are only deducted for successfully processed API calls.
  </Accordion>
  <Accordion title="What happens when my balance runs out?">
    You receive a `402 Payment Required` response. No requests are processed until you top up your balance. There are no overages or surprise charges.
  </Accordion>
  <Accordion title="How do I check my remaining balance?">
    Your current credit balance is always visible in the [Dashboard](https://www.apitwitter.com/dashboard). It updates in real time after each API call.
  </Accordion>
</AccordionGroup>
