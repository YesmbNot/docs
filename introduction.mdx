## Quick Start

<Steps>
  <Step title="Create an Account">
    Register at [apitwitter.com](https://apitwitter.com/register) to get your API key.
  </Step>
  <Step title="Add Credits">
    Top up from **$10** via Stripe (card) or CryptoCloud (crypto). 1 USD = 100,000 credits.
  </Step>
  <Step title="Make API Calls">
    Include your API key, Twitter cookies, and proxy with every request.
  </Step>
</Steps>

## Base URL

All requests go to:

```
https://api.apitwitter.com
```

## Requirements

| Requirement | Description |
|------------|-------------|
| **API Key** | Your `X-API-Key` header (starts with `tda_`) |
| **Cookies** | Your Twitter session cookies (`ct0` and `auth_token`) |
| **Proxy** | Your own proxy URL (HTTP or SOCKS5) |

<Warning>
We do not provide proxies or cookies. You must supply your own with every request.
</Warning>
