---
title: "Get Tweets by IDs"
description: "GET multiple tweets by IDs in a single ApiTwitter pool call. Batch tweet lookup with full tweet data. No cookies needed."
openapi: "GET /twitter/tweets/lookup"
---

<Info>
This endpoint uses the server-side cookie/proxy pool. Pass tweet IDs as a `tweet_ids` query parameter. No cookie or proxy required.
</Info>
