---
title: "Batch Get Users"
description: "Batch lookup multiple Twitter users by IDs in a single ApiTwitter API call. Server-side pool, no cookies or proxy required."
openapi: "GET /twitter/users/batch"
---

<Info>
This endpoint uses the server-side cookie/proxy pool. Pass user IDs as a `userIds` query parameter. No cookie or proxy required.
</Info>
