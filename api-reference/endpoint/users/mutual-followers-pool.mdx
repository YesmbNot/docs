---
title: "Get Mutual Followers"
description: "GET mutual followers between your session and a Twitter user via ApiTwitter pool. Paginated results, no cookies required."
openapi: "GET /twitter/user/{userName}/followers-you-know"
---

<Info>
This endpoint uses the server-side cookie/proxy pool. You only need your API key. Use `count` and `cursor` query parameters for pagination.
</Info>
