## GET /twitter/user/{userName}

Get a user's profile information by their screen name.

**Cost:** 16 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| userName | string | path | Yes | Twitter screen name |
| cookie | string | query | Yes | Twitter auth cookie |
| proxy | string | query | No | Proxy URL to use for the request |

<CodeGroup>
```bash cURL
curl -X GET "https://api.apitwitter.com/twitter/user/elonmusk?cookie=YOUR_COOKIE&proxy=YOUR_PROXY" \
  -H "X-API-Key: YOUR_API_KEY"
```
</CodeGroup>

## GET /twitter/user/{userName}/followers

Get a paginated list of a user's followers.

**Cost:** 14 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| userName | string | path | Yes | Twitter screen name |
| cookie | string | query | Yes | Twitter auth cookie |
| proxy | string | query | No | Proxy URL to use for the request |
| pageSize | integer | query | No | Number of results per page (default: 200) |
| cursor | string | query | No | Pagination cursor for the next page |

<CodeGroup>
```bash cURL
curl -X GET "https://api.apitwitter.com/twitter/user/elonmusk/followers?cookie=YOUR_COOKIE&pageSize=100&cursor=NEXT_CURSOR" \
  -H "X-API-Key: YOUR_API_KEY"
```
</CodeGroup>

## POST /twitter/user/{user_id}/remove-follower

Remove a follower from your account.

**Cost:** 225 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| user_id | string | path | Yes | Numeric Twitter user ID of the follower to remove |
| cookie | string | body | Yes | Twitter auth cookie |
| proxy | string | body | No | Proxy URL to use for the request |

<CodeGroup>
```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/user/44196397/remove-follower" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cookie": "YOUR_COOKIE",
    "proxy": "YOUR_PROXY"
  }'
```
</CodeGroup>
