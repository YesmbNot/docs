## POST /twitter/tweets/{tweet_id}/like

Like a tweet.

**Cost:** 225 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| tweet_id | string | path | Yes | ID of the tweet to like |
| cookie | string | body | Yes | Twitter auth cookie |
| proxy | string | body | No | Proxy URL to use for the request |

<CodeGroup>
```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/tweets/1234567890/like" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cookie": "YOUR_COOKIE",
    "proxy": "YOUR_PROXY"
  }'
```
</CodeGroup>
