## POST /twitter/timeline/for-you

Retrieve the algorithmic "For You" home timeline.

**Cost:** 14 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| cookie | string | body | Yes | Twitter auth cookie |
| proxy | string | body | No | Proxy URL to use for the request |
| count | integer | body | No | Number of tweets to return (default: 20) |
| cursor | string | body | No | Pagination cursor for the next page |

<CodeGroup>
```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/timeline/for-you" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cookie": "YOUR_COOKIE",
    "proxy": "YOUR_PROXY",
    "count": 20
  }'
```
</CodeGroup>
