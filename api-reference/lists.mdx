## POST /twitter/lists

Create a new Twitter list.

**Cost:** 225 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| name | string | body | Yes | Name of the list |
| description | string | body | No | Description of the list |
| is_private | boolean | body | No | Whether the list is private (default: false) |
| cookie | string | body | Yes | Twitter auth cookie |
| proxy | string | body | No | Proxy URL to use for the request |

<CodeGroup>
```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/lists" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Tech Leaders",
    "description": "Top voices in tech",
    "is_private": false,
    "cookie": "YOUR_COOKIE",
    "proxy": "YOUR_PROXY"
  }'
```
</CodeGroup>
