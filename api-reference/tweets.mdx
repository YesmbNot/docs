## GET /twitter/tweets

Retrieve one or more tweets by their IDs.

**Cost:** 14 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| tweet_ids | string | query | Yes | Comma-separated list of tweet IDs |
| cookie | string | query | Yes | Twitter auth cookie |
| proxy | string | query | No | Proxy URL to use for the request |

<CodeGroup>
```bash cURL
curl -X GET "https://api.apitwitter.com/twitter/tweets?tweet_ids=1234567890,9876543210&cookie=YOUR_COOKIE" \
  -H "X-API-Key: YOUR_API_KEY"
```
</CodeGroup>

## GET /twitter/user/{userName}/tweets

Get a paginated list of tweets posted by a user.

**Cost:** 14 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| userName | string | path | Yes | Twitter screen name |
| cookie | string | query | Yes | Twitter auth cookie |
| proxy | string | query | No | Proxy URL to use for the request |
| cursor | string | query | No | Pagination cursor for the next page |

<CodeGroup>
```bash cURL
curl -X GET "https://api.apitwitter.com/twitter/user/elonmusk/tweets?cookie=YOUR_COOKIE&cursor=NEXT_CURSOR" \
  -H "X-API-Key: YOUR_API_KEY"
```
</CodeGroup>

## POST /twitter/tweets

Create a new tweet or reply to an existing tweet.

**Cost:** 225 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| tweet_text | string | body | Yes | The text content of the tweet |
| cookie | string | body | Yes | Twitter auth cookie |
| proxy | string | body | No | Proxy URL to use for the request |
| reply_to_tweet_id | string | body | No | Tweet ID to reply to |

<CodeGroup>
```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/tweets" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "tweet_text": "Hello from the API!",
    "cookie": "YOUR_COOKIE",
    "proxy": "YOUR_PROXY"
  }'
```
</CodeGroup>
