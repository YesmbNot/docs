## POST /twitter/tweets/{tweet_id}/bookmark

Add a tweet to your bookmarks.

**Cost:** 225 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| tweet_id | string | path | Yes | ID of the tweet to bookmark |
| cookie | string | body | Yes | Twitter auth cookie |
| proxy | string | body | No | Proxy URL to use for the request |

<CodeGroup>
```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/tweets/1234567890/bookmark" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cookie": "YOUR_COOKIE",
    "proxy": "YOUR_PROXY"
  }'
```
</CodeGroup>

## GET /twitter/bookmarks

Get a paginated list of your bookmarked tweets.

**Cost:** 14 credits

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| X-API-Key | string | header | Yes | Your API key |
| cookie | string | query | Yes | Twitter auth cookie |
| proxy | string | query | No | Proxy URL to use for the request |
| cursor | string | query | No | Pagination cursor for the next page |

<CodeGroup>
```bash cURL
curl -X GET "https://api.apitwitter.com/twitter/bookmarks?cookie=YOUR_COOKIE&cursor=NEXT_CURSOR" \
  -H "X-API-Key: YOUR_API_KEY"
```
</CodeGroup>
