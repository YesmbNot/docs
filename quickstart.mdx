## Prerequisites

Before making your first request, you need:

1. An ApiTwitter account — [register here](https://apitwitter.com/register)
2. Credits on your balance — minimum $10 top-up

For **GET** read endpoints, that's all you need. For **POST** endpoints (which give you full control with your own cookies), you also need:

3. Your Twitter cookies (`ct0` and `auth_token`)
4. A proxy URL

## Step 1: Get Your API Key

After registration, your API key is automatically created. Find it in your [Dashboard → API Key](https://apitwitter.com/api-keys) page.

Your key looks like: `tda_k8x2mN3s_...`

## Step 2: Get Your Twitter Cookies

1. Open [x.com](https://x.com) and log in
2. Open DevTools (F12) → **Application** → **Cookies** → `https://x.com`
3. Copy the values of `ct0` and `auth_token`
4. Format them as: `ct0=YOUR_CT0;auth_token=YOUR_AUTH_TOKEN`

## Step 3: Make Your First Request

### Option A: GET (simplest — uses server pool)

<CodeGroup>

```bash cURL
curl "https://api.apitwitter.com/twitter/user/elonmusk" \
  -H "X-API-Key: tda_your_key_here"
```

```python Python
import requests

response = requests.get(
    "https://api.apitwitter.com/twitter/user/elonmusk",
    headers={"X-API-Key": "tda_your_key_here"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://api.apitwitter.com/twitter/user/elonmusk",
  { headers: { "X-API-Key": "tda_your_key_here" } }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

### Option B: POST (your own cookies & proxy)

<CodeGroup>

```bash cURL
curl -X POST "https://api.apitwitter.com/twitter/user/elonmusk" \
  -H "X-API-Key: tda_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"cookie": "ct0=abc123;auth_token=xyz789", "proxy": "http://user:pass@proxy:8080"}'
```

```python Python
import requests

response = requests.post(
    "https://api.apitwitter.com/twitter/user/elonmusk",
    headers={"X-API-Key": "tda_your_key_here"},
    json={
        "cookie": "ct0=abc123;auth_token=xyz789",
        "proxy": "http://user:pass@proxy:8080"
    }
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://api.apitwitter.com/twitter/user/elonmusk",
  {
    method: "POST",
    headers: {
      "X-API-Key": "tda_your_key_here",
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      cookie: "ct0=abc123;auth_token=xyz789",
      proxy: "http://user:pass@proxy:8080"
    })
  }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>
